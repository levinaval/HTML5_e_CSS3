Neste site https://www.w3schools.com/cssref/css_websafe_fonts.php podemos pesquisar as melhores fontes seguras da Web para HTML e CSS

Para não termos problemas com os dispositivos que acessam o nosso site, temos que escrever no codigo na parte de style " font-family: Arial, Helvetica, sans-serif;". desta forma todos os dispositivos reconheceram a fonte a ser exibida!
