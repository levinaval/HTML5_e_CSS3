Para definir uma fonte, temos que conhecer alguns detalhes, Que são os itens metricos. Altura-x: Esta métrica é usada para definir a altura das letras minúsculas. Altura das maiúsculas: Define o espaço das letras maiúsculas. Vazamento ascendente: Define o espaço que as letras minúsculas vazão para cima. Vazamento descendente: Define o espaço que as letras minúsculas vazão para baixo. Corpo: Define espaço somando todas as alturas das letras.
