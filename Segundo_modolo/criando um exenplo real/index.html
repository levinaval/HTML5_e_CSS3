Configurações globais da Css

* { font-family: Arial, Helvetica, sans-serif; } - O asterisco serve para as configurações globais. tipo de fontes, se as cores ficarão em 100% da imagem, etc.

body { background-image: linear-gradient(to right, #C3C6CB, #A3916D, #9D9C9A); } - É para justificar o corpo do site

main { background-color: white; border-radius: 10px; box-shadow: 5px 5px 15px #A3916D; width: 600px; padding: 10px; margin: auto; } - É para justificar o texto que ficará dentro da Main {} com cor, largura, acabamento, centralizar, definir bordas e sombreamento: primeiro parâmetro - Quanto a sombra vai andar para o lado, segundo parâmetro - Quanto a sombra vai andar para baixo, terceiro parâmetro - Quanto a sombra vai andar espalhar e o quarto parâmetro - a transparência.

Falando sobre o H1

h1 { color: #9ea0a4; text-align: center; text-shadow: 1px 1px 2px #A3916D; }

h2 { color: #A3916D; } - É para justificar os títulos, colocando cor, o lugar em que o texto ficará na página do site, e colocar um sombreamento.

Nunca desistir

p { text-align: justify; color: black; } - É para justificar os parágrafos. com cor, tipo de texto, etc.
